== Description ==
Просто й плагин что позволит вам публиковать новости или другие статьи в вашу группу телеграм


== Features ==

- добавляет новые статьи
- обновляет сообщение в чате если обновить на сайте 
- удаляет


== Installation ==

1- Install and activate
2- In the settings, enter the token and id of your chat.
3- in the forms below indicate which ones you want to receive in telegrams.

more details on the site

https://pechenki.top/plugin-and-modules/125-tel-publish.html
